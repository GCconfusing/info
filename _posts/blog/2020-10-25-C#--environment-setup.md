## 1.下载Visual Studio
https://visualstudio.microsoft.com/zh-hans/vs/
版本选择(Community)社区版

## 2. 安装Visual Studio
开发环境选择.Net开发

## 3.注册微软账号
在微软官网注册,如果有的话直接登录

## 4.登录并且启动Visual Studio
创建一个控制台应用程序项目

## 5.测试
输入下列代码运行
```
using System;

using System;
namespace HelloWorldApplication
{
    class HelloWorld
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello World!");
            Console.ReadKey();
        }
    }
}
```
如果输出了Hello World则成功配置环境
